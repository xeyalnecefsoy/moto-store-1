# MotoStore (Frontend-only)

- **Stack**: Next.js 14 (App Router) + TypeScript + Tailwind CSS
- **Theme**: Dark UI with bright green primary accents

## Getting Started

1. Install dependencies
   ```bash
   npm install
   ```
2. Run the dev server
   ```bash
   npm run dev
   ```
3. Open http://localhost:3000

## Pages

- Home: Hero, categories, featured bikes
- Products: Grid list of motorcycles
- Product Detail: Specs + Add to Cart (placeholder)

## Notes

- This is frontend-only. Cart is a placeholder.
- Tailwind is configured via `tailwind.config.ts` and `src/app/globals.css`.
